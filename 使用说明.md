## 🚀 快速开始（3步上手）

### 第1步：准备环境

```bash
# 仅需安装 gTTS（Python 3.9 原生兼容）
pip3 install gtts

# 编辑配置文件，填入你的API密钥
cp 01_核心脚本/smart_hybrid.py .  # 复制到根目录使用
# 修改文件内的 MOONSHOT_API_KEY 和 REPLICATE_API_TOKEN
```

### 第2步：准备文章

```bash
# 创建文章文件
echo "你的公众号文章内容" > article.md
```

### 第3步：一键生成

```bash
# 生成完整视频方案
python3 01_核心脚本/smart_hybrid.py article.md

# 进入生成的项目目录
cd 项目输出/2026-04-26_article_hybrid/

# 生成配音
python3 ../../01_核心脚本/tts_gtts.py --from-project plan.json
```

## 📋 完整工作流

```
┌─────────────────────────────────────────────────────────────┐
│  1. 文案准备 → 2. AI生成方案 → 3. 图片生成 → 4. 配音 → 5. 剪辑  │
└─────────────────────────────────────────────────────────────┘

详细步骤：
1. 复制公众号文章到 markdown 文件
2. 运行 smart_hybrid.py 生成视频方案（plan.json + 分镜脚本）
3. 按提示生成图片（API自动 + 手动补充）
4. 运行 tts_gtts.py 生成配音
5. 用剪映导入素材，按脚本剪辑
```

## 🎯 核心脚本说明

### smart_hybrid.py - 智能视频方案生成
```bash
python3 01_核心脚本/smart_hybrid.py <文章.md>

# 功能：
# - 调用 Kimi API 分析文章
# - 生成视频分镜脚本
# - 智能分配API/手动图片生成
# - 创建项目文件夹结构
```

### tts_gtts.py - AI配音生成
```bash
# 从项目生成完整配音
python3 01_核心脚本/tts_gtts.py --from-project 项目输出/XXX/plan.json

# 单条测试
python3 01_核心脚本/tts_gtts.py --script "测试文案" --output test.mp3
```

## ⚙️ API配置

在 `smart_hybrid.py` 中配置你的API密钥：

```python
MOONSHOT_API_KEY = "sk-your-kimi-api-key"      # Kimi API
REPLICATE_API_TOKEN = "r8-your-replicate-token" # Replicate（可选）
```

获取方式：
- Kimi API: https://platform.moonshot.cn/
- Replicate: https://replicate.com/（新用户有$5免费额度）

## ⚠️ 注意事项

1. **Python版本**: 当前使用 Python 3.9 + gTTS（无需升级）
2. **API配额**: 免费额度有限，配额跟踪见 `项目输出/quota_tracker.json`
3. **网络**: gTTS需要访问Google服务，可能需要网络代理
4. **配音语速**: gTTS中文语速适中，如需调整可改用Edge-TTS（需Python 3.11）
