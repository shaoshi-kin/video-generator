# 快速开始 - 3步生成视频

**版本**: 3.0 | **更新日期**: 2025-04-27

---

## 最快路径（30秒上手）

```bash
# 1. 创建项目（以图片模式为例）
python3 01_核心脚本/video_generator_pro.py --init --template news
# 输入项目名，回车，项目创建在 projects/你的项目名/

# 2. 替换素材
#   - 改文案: projects/你的项目名/01_article/文章.md
#   - 换图片: projects/你的项目名/03_images/
#   - 换水印: projects/你的项目名/watermark.png（直接替换图片即可）
#   - 加BGM:  projects/你的项目名/02_bgm/（可选）
#   - 加音效: projects/你的项目名/02_sfx/（可选，转场时自动混入）

# 3. 生成视频（所有参数从配置读取，无需长命令）
python3 01_核心脚本/video_generator_pro.py -p projects/你的项目名
# 输出: projects/你的项目名/07_final/你的项目名_YYYYMMDD_HHMMSS.mp4
```

## 新功能一览（v3.0）

| 功能 | 说明 | 配置位置 |
|------|------|----------|
| **默认字幕** | 自动从文章生成，无需 `--subtitle` | `.video_config.json` |
| **字幕动画** | 上滑进入 / 淡入效果 | `subtitle_animation` |
| **场景淡入淡出** | 每个片段开头结尾自然过渡 | `scene_fade` |
| **28种转场** | fade、pixelize、slideleft 等 | `transition` |
| **水印/Logo** | 自动叠加，替换 PNG 即可 | `watermark` |
| **转场音效** | 自动读取 `02_sfx/` 混入 | `sfx` |
| **配置优先** | 所有参数写进 `.video_config.json` | 初始化自动生成 |

## 方式一：图片 + AI配音（推荐新手）

### 第1步：创建文件夹

```bash
mkdir -p projects/我的视频/{01_article,03_images}
```

### 第2步：放入素材

**写文章** `projects/我的视频/01_article/文章.md`：
```markdown
# 标题

@全局:女声
@默认图: 吃饭

@新闻男: 大家好，这是第一段，用默认图吃饭。

@图:睡觉
@新闻男: 这是第二段，强制使用睡觉图片。

回到女声和默认图吃饭。
（空行后自动回到全局默认：女声 + 吃饭）
```

**放图片** 到 `projects/我的视频/03_images/`：
- `吃饭.jpg`（默认图片）
- `睡觉.png`（第二段用）

**支持中文文件名！** 直接用 `@图:吃饭` 引用。

### 第3步：运行命令

```bash
cd /Users/kingshaoshi/Desktop/claudeCode
python3 01_核心脚本/video_generator_pro.py -p projects/我的视频
```

**完成！** 视频在 `projects/我的视频/07_final/`

## 核心规则速查

### 标记规范

| 标记 | 作用 | 示例 |
|------|------|------|
| `@全局:音色` | 设置全局默认音色 | `@全局:女声` |
| `@默认图:名称` | 设置全局默认图片 | `@默认图:吃饭` |
| `@图:名称` | 指定当前段落图片 | `@图:睡觉` |
| `@音色:内容` | 指定当前段落音色 | `@男声: 你好` |

### 空行重置规则

**空行后，音色和图片都回到全局默认值！**

```markdown
@全局:女声
@默认图: 吃饭

@新闻男: 第一段，新闻男 + 吃饭。

第二段，女声 + 吃饭（空行后重置）。

@图:睡觉
@云夏: 第三段，云夏 + 睡觉。

第四段，女声 + 吃饭（空行后重置）。
```

### 图片命名（支持中文！）

```
03_images/
├── 吃饭.jpg       ← @图:吃饭 或 @默认图:吃饭
├── 睡觉.png       ← @图:睡觉
├── 打豆豆.jpg     ← @图:打豆豆
└── 背景.jpg       ← @默认图:背景
```

## 配置优先（推荐）

**无需长命令行！** 所有参数写进 `.video_config.json`：

```json
{
  "subtitle": true,
  "subtitle_style": "youtube",
  "subtitle_animation": "slide_up",
  "scene_fade": 0.3,
  "transition": "pixelize",
  "watermark": "watermark.png",
  "watermark_position": "bottom-right",
  "bgm_volume": 0.25,
  "sfx": false,
  "rate": "+18%"
}
```

改完配置，直接运行：
```bash
python3 01_核心脚本/video_generator_pro.py -p projects/我的视频
```

临时覆盖（优先级高于配置）：
```bash
python3 01_核心脚本/video_generator_pro.py -p projects/我的视频 \
    --transition fade \
    --scene-fade 0.5
```

## 常见问题

### Q: 为什么空行后音色变了？
**A**: 空行会重置音色为 `@全局` 设置的默认值。如果不想重置，把内容放在同一行，不要加空行。

### Q: 支持中文图片名吗？
**A**: 支持！`吃饭.jpg` 可以用 `@图:吃饭` 引用。

### Q: Windows 创建的 .md 文件能用吗？
**A**: 支持！自动处理 Windows CRLF 换行符。

### Q: 修改了水印/字幕配置但没生效？
**A**: `06_scenes/` 下的旧场景文件不带新效果，删除它们重新生成：
```bash
rm projects/我的视频/06_scenes/*.mp4
```

### Q: 如何关闭字幕？
**A**: 改 `.video_config.json` 里的 `"subtitle": false`，或命令行：
```bash
python3 01_核心脚本/video_generator_pro.py -p projects/我的视频 --no-subtitle
```
