**版本**: 2.0 | **更新日期**: 2025-04-26

## 方式一：图片 + AI配音（推荐新手）

### 第1步：创建文件夹

```bash
mkdir -p projects/我的视频/{01_article,02_images}
```

### 第2步：放入素材

**写文章** `projects/我的视频/01_article/文章.md`：
```markdown
# 标题

@全局:女声
@默认图: 吃饭

@新闻男: 大家好，这是第一段，用默认图吃饭。

@图:睡觉
@新闻男: 这是第二段，强制使用睡觉图片。

回到女声和默认图吃饭。
（空行后自动回到全局默认：女声 + 吃饭）
```

**放图片** 到 `projects/我的视频/02_images/`：
- `吃饭.jpg`（默认图片）
- `睡觉.png`（第二段用）

**支持中文文件名！** 直接用 `@图:吃饭` 引用。

### 第3步：运行命令

```bash
cd /Users/kingshaoshi/Desktop/claudeCode
python3 01_核心脚本/video_generator_pro.py -p projects/我的视频 --voice Xiaoxiao
```

**完成！** 视频在 `projects/我的视频/04_final/final_video_pro.mp4`

## 进阶：添加字幕

加 `--subtitle` 参数：

```bash
python3 01_核心脚本/video_generator_pro.py \
    -p projects/我的视频 \
    --voice Xiaoxiao \
    --subtitle \
    --subtitle-style news
```

字幕风格可选：`news`（新闻）/ `youtube` / `tiktok`（抖音）/ `minimal`

## 核心规则速查

### 标记规范

| 标记 | 作用 | 示例 |
|------|------|------|
| `@全局:音色` | 设置全局默认音色 | `@全局:女声` |
| `@默认图:名称` | 设置全局默认图片 | `@默认图:吃饭` |
| `@图:名称` | 指定当前段落图片 | `@图:睡觉` |
| `@音色:内容` | 指定当前段落音色 | `@男声: 你好` |

### 空行重置规则

**空行后，音色和图片都回到全局默认值！**

```markdown
@全局:女声
@默认图: 吃饭

@新闻男: 第一段，新闻男 + 吃饭。

第二段，女声 + 吃饭（空行后重置）。

@图:睡觉
@云夏: 第三段，云夏 + 睡觉。

第四段，女声 + 吃饭（空行后重置）。
```

### 图片命名（支持中文！）

```
02_images/
├── 吃饭.jpg       ← @图:吃饭 或 @默认图:吃饭
├── 睡觉.png       ← @图:睡觉
├── 打豆豆.jpg     ← @图:打豆豆
└── 背景.jpg       ← @默认图:背景
```

## 常见问题

### Q: 为什么空行后音色变了？
**A**: 空行会重置音色为 `@全局` 设置的默认值。如果不想重置，把内容放在同一行，不要加空行。

### Q: 支持中文图片名吗？
**A**: 支持！`吃饭.jpg` 可以用 `@图:吃饭` 引用。

### Q: Windows 创建的 .md 文件能用吗？
**A**: 支持！自动处理 Windows CRLF 换行符。

### Q: 冒号后要加空格吗？
**A**: 都可以。`@图:吃饭` 和 `@图: 吃饭` 效果相同。
