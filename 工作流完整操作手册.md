## 二、两种工作模式详解

### 2.1 图片模式（Ken Burns动画 + 转场特效）

**适用场景**: 文章配图、PPT讲解、静态图片展示

**特点**:
- 文章每段落生成一段音频
- 图片自动匹配音频段落数
- Ken Burns 慢速缩放动画
- 场景淡入淡出（可配置时长）
- 段落间转场特效（28种可选）
- 转场音效自动混入（可选）
- 最高质量编码（veryslow + crf 15）

**运行流程**:
```
01_article/文章.md → edge-tts生成 → 05_audio/scene_01~N.mp3
03_images/图片 + 音频 + 字幕 + 水印 + 淡入淡出 → 场景片段 → 06_scenes/
06_scenes/ + 转场特效 + BGM → 07_final/最终视频
```

### 2.2 视频模式（原画画质保留）

**适用场景**: 实拍素材配音、视频混剪、保留原始画质

**特点**:
- 整篇文章合并为一个音频
- 视频直接复制（`-c:v copy`）不损失画质
- 音频驱动时长（视频自动循环）
- AI配音替换原视频音频
- 字幕/水印/BGM 同样支持

**运行流程**:
```
01_article/文章.md → edge-tts生成 → 05_audio/scene_01.mp3（完整版）
04_videos/原视频 + 完整音频 + 字幕 + 水印 → 合成视频
```

## 三、文章标记规范（v5.0）

### 3.1 核心规则

| 标记 | 作用 | 格式 | 示例 |
|------|------|------|------|
| `@全局:音色` | 设置全局默认音色 | `@全局:女声` | `@全局:Yunyang` |
| `@默认图:名称` | 设置全局默认图片 | `@默认图:吃饭` | `@默认图: 背景` |
| `@图:名称` | 指定当前段落图片 | `@图:睡觉` | `@图: 数据` |
| `@音色:内容` | 指定当前段落音色 | `@男声:你好` | `@新闻男: 大家好` |

**重要规则**:
- **空行分隔段落** - 每段以空行结束
- **空行重置** - 空行后音色和图片都回到全局默认值
- **标记位置** - `@图:` 和 `@音色:` 放在段落第一行
- **冒号后空格** - 支持 `@图: 吃饭` 或 `@图:吃饭`
- **中文文件名** - 图片名直接用中文，如 `吃饭.jpg`

### 3.2 使用示例

#### 示例1：基础用法
```markdown
# 文章标题

@全局:女声
@默认图: 吃饭

@新闻男: 这是第一段，使用默认图吃饭。

@图:睡觉
@新闻男: 这是第二段，强制使用睡觉图片。

这段继续用默认图片，女声。
（空行后 → 女声 + 吃饭）
```

**效果**:
- 第1段：新闻男 + 吃饭
- 第2段：新闻男 + 睡觉（指定）
- 第3段：女声 + 吃饭（空行后重置）

#### 示例2：复杂场景
```markdown
@全局:女声
@默认图: 背景

@男声: 开场白，男声 + 默认图背景。

@图:图表
@男声: 展示数据，男声 + 图表。

回到女声讲解，图片回到默认背景。
（空行后 → 女声 + 背景）

@图:产品
@女声: 介绍产品，女声 + 产品图。

@图:价格
@男声: 公布价格，男声 + 价格图。

结尾总结，回到女声和默认图。
```

## 五、完整使用教程

### 5.1 创建新项目（推荐 --init）

#### 方式一：命令行初始化（自动生成配置和示例）

```bash
# 图片模式项目（带示例文章和图片目录）
python3 01_核心脚本/video_generator_pro.py --init --template news

# 视频模式项目
python3 01_核心脚本/video_generator_pro.py --init --template video
```

执行后会：
1. 创建项目文件夹结构
2. 生成 `.video_config.json`（带默认配置）
3. 生成示例 `文章.md`
4. 生成 `参数说明.md`
5. 生成默认 `watermark.png`

#### 方式二：手动创建

```bash
# 图片模式项目
mkdir -p projects/my_article/{01_article,03_images}

# 视频模式项目
mkdir -p projects/my_video/{01_article,04_videos}
```

### 5.2 命令行参数

```bash
python3 01_核心脚本/video_generator_pro.py \
    -p projects/my_project \           # 项目路径（必需）
    --voice Xiaoxiao \                 # AI音色（默认从配置读取）
    --rate +18% \                      # 语速（默认从配置读取）
    --subtitle \                       # 添加字幕（默认从配置读取）
    --subtitle-style youtube \         # 字幕风格（默认从配置读取）
    --subtitle-animation slide_up \    # 字幕动画（默认从配置读取）
    --scene-fade 0.3 \                 # 场景淡入淡出时长（默认从配置读取）
    --transition pixelize \            # 转场效果（默认从配置读取）
    --transition-duration 0.8 \        # 转场时长（默认从配置读取）
    --resolution 1920x1080 \           # 分辨率（默认从配置读取）
    --fps 30 \                         # 帧率（默认从配置读取）
    --watermark watermark.png \        # 水印路径（默认从配置读取）
    --watermark-position bottom-right \# 水印位置（默认从配置读取）
    --bgm-volume 0.25 \                # BGM音量（默认从配置读取）
    --sfx \                            # 开启转场音效（默认从配置读取）
    --batch \                          # 批量处理多个项目
```

### 5.3 配置优先（推荐）

**无需长命令行！** 所有参数写进 `.video_config.json`：

```json
{
  "voice": "Xiaoxiao",
  "resolution": "1920x1080",
  "fps": 30,
  "subtitle": true,
  "subtitle_style": "youtube",
  "subtitle_animation": "slide_up",
  "scene_fade": 0.3,
  "transition": "pixelize",
  "transition_duration": 0.8,
  "rate": "+18%",
  "bgm_volume": 0.25,
  "watermark": "watermark.png",
  "watermark_position": "bottom-right",
  "sfx": false
}
```

改完配置，直接运行：
```bash
python3 01_核心脚本/video_generator_pro.py -p projects/我的项目
```

临时覆盖（优先级高于配置）：
```bash
python3 01_核心脚本/video_generator_pro.py -p projects/我的项目 \
    --transition fade \
    --scene-fade 0.5
```

每个场景片段开头和结尾自动添加淡入淡出效果：
- `scene_fade: 0.3` → 开头淡入0.3秒，结尾淡出0.3秒
- `scene_fade: 0` → 关闭淡入淡出
- 推荐值：0.2 ~ 0.5

### 6.4 水印/Logo

- 默认启用，替换 `watermark.png` 即可更换
- 支持位置：`top-left`, `top-right`, `bottom-left`, `bottom-right`, `center`
- 默认透明度：0.75
- 水印路径支持相对路径（基于项目目录）或绝对路径

### 6.5 转场音效（SFX）

- 开启后自动读取 `02_sfx/` 目录下的音效文件
- 在转场时自动混入
- 多个音效文件会随机选择
- 适合添加 whoosh、切换等音效增强节奏感

### 6.6 编码参数

#### 图片模式（最高质量）
```python
-c:v libx264        # H.264编码
-preset veryslow    # 最慢预设 = 最佳质量
crf 15              # 接近无损画质
-tune stillimage    # 针对静态图片优化
-b:a 256k           # 高码率音频
-movflags +faststart # 网络播放优化
```

#### 视频模式（画质保留）
```python
-c:v copy           # 直接复制视频流，零损失
-c:a aac -b:a 192k  # 音频重新编码
-map 0:v:0 -map 1:a:0  # 视频+新音频
-stream_loop N      # 视频循环（音频更长时）
```

## 七、常见问题

### Q1: 为什么空行后音色变了？
**A**: 空行会重置音色为 `@全局` 设置的默认值。如果不想重置，把内容放在同一行，不要加空行。

### Q2: 支持中文图片名吗？
**A**: 支持！`吃饭.jpg` 可以用 `@图:吃饭` 引用。

### Q3: Windows 创建的 .md 文件能用吗？
**A**: 支持！自动处理 Windows CRLF 换行符。

### Q4: 修改了水印/字幕配置但没生效？
**A**: `06_scenes/` 下的旧场景文件不带新效果，删除它们重新生成：
```bash
rm projects/我的视频/06_scenes/*.mp4
```

### Q5: 如何关闭字幕？
**A**: 改 `.video_config.json` 里的 `"subtitle": false`，或命令行：
```bash
python3 01_核心脚本/video_generator_pro.py -p projects/我的视频 --no-subtitle
```

### Q6: 图片数量和音频段落不匹配？
**A**: 图片不够时会复用已有图片。也可以通过 `@图:` 指定复用某张图。

### Q7: 视频比音频长怎么办？
**A**: 自动以音频时长为准，视频会截断到音频结束。

### Q8: 音频比视频长怎么办？
**A**: 自动计算循环次数，使用 `-stream_loop` 让视频循环播放直到音频结束。

### Q9: 如何只生成音频不合成视频？
**A**: 运行后按 Ctrl+C 中断，或单独使用 article_to_voice_edge.py：
```bash
python3 01_核心脚本/article_to_voice_edge.py 文章.md --voice Xiaoxiao --output output.mp3
```

### Q10: 已有音频如何跳过生成？
**A**: 只要 `05_audio/` 目录存在且包含 `.mp3` 文件，就会自动跳过音频生成步骤。

### Q11: 为什么音色没按我写的来？
**A**: 检查是否有空行。空行会重置音色为 `@全局` 设置的默认值。如果不想重置，把内容放在同一行：
```markdown
@新闻男: 这是第一段。
@新闻男: 这是第二段。  # 不要空行，音色保持新闻男
```

### 8.1 视频合成失败
