# 视频生成器 Pro - 产品拓展规划

**日期**: 2026-04-28
**目标**: 从"编辑器"升级为"AI 内容工厂"

---

## 一、产品定位重新定义

| 维度 | 剪映专业版 | 我们的方向 |
|------|-----------|-----------|
| **输入** | 素材 + 文案（人准备） | 一个标题/关键词 |
| **过程** | 人找图、写稿、剪视频 | AI 全自动 |
| **输出** | 1 个视频/小时 | 10 个视频/小时 |
| **用户** | 创作者、剪辑师 | 自媒体矩阵、营销号、资讯站 |

**核心差异**: 剪映是编辑器（人做一切），我们是内容工厂（人只给主题）。

---

## 二、三阶段拓展路线

### 第一阶段：标题 → 文章 ✅ 已完成

**功能**: 输入标题，AI 自动搜索资料、总结、生成口播文章

**命令**:
```bash
python3 01_核心脚本/video_generator_pro.py --auto-article "马斯克收购推特内幕"
```

**技术栈**:
- 搜索: DuckDuckGo 免费搜索（无需 API Key）
- 总结: DeepSeek API (默认) / Kimi API
- 输出: 01_article/文章.md

**流程**:
```
标题 → DuckDuckGo 搜索 → 取前5条结果 → LLM 阅读总结 → 生成口播稿 → 保存
```

**状态**: ✅ 已完成

---

### 第二阶段：文章 → 自动配图 ✅ 已完成

**功能**: 每段文字自动匹配图片

**命令**:
```bash
python3 01_核心脚本/video_generator_pro.py -p projects/xxx --auto-images
```

**技术栈**:
- 关键词提取: LLM API 或零依赖规则提取
- 图片来源:
  - Pollinations.ai（免费 AI 生图，无需 Key）
  - Unsplash API（50次/小时免费）
  - Pexels API（200次/小时免费）

**流程**:
```
文章分段 → 提取关键词 → 调用图库 API → 下载图片 → 自动插入 @图:xxx 标记
```

**状态**: ✅ 已完成

---

### 第三阶段：标题 → 视频（终极一键） ✅ 已完成

**功能**: 全流程自动化，输入标题出视频

**命令**:
```bash
python3 01_核心脚本/video_generator_pro.py --factory "马斯克收购推特内幕"
```

**流程**:
```
标题 → 搜索资料 → 生成文章 → 自动配图 → AI 配音 → 合成视频 → 生成发布文案
```

**状态**: ✅ 已完成

## 四、待开发功能

### 批量矩阵号 `--variants N`

一个标题生成 N 个不同版本（不同角度/不同钩子开头），适合同一热点做多账号分发。

```bash
# 一个标题，生成 5 个不同版本
python3 01_核心脚本/video_generator_pro.py --factory "热点标题" --variants 5
```

**状态**: ⏳ 待开发

### 文章改写/去重

输入一篇别人的文章，AI 改写成自己的口播稿。

```bash
python3 01_核心脚本/video_generator_pro.py --rewrite article.md
```

**状态**: ⏳ 待开发
