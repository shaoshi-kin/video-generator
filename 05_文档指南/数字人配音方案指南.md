## 🏠 本地部署（方案B：ChatTTS）

### 硬件要求

- **显存**: 8GB+（NVIDIA显卡）
- **内存**: 16GB+
- **硬盘**: 10GB空间

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/2noise/ChatTTS.git ~/ChatTTS
cd ~/ChatTTS

# 2. 安装依赖
pip install -r requirements.txt

# 3. 下载模型（自动）
python -c "import ChatTTS; ChatTTS.Chat().load()"
```

### 使用ChatTTS

```bash
python3 tts_generator.py --script "你的文案" --engine chattts --output voice.mp3
```

**ChatTTS优势**：
- 声音极其自然，接近真人
- 支持情感控制（开心、严肃、悲伤）
- 支持语速调节

## 💰 付费方案（方案C：ElevenLabs）

### 价格

| 套餐 | 价格 | 字符数 | 适用 |
|------|------|--------|------|
| 免费版 | $0 | 10K/月 | 测试 |
| 基础版 | $5/月 | 100K/月 | 个人创作者 |
| 专业版 | $22/月 | 500K/月 | 工作室 |

### 接入方法

```python
import requests

def elevenlabs_tts(text, api_key, voice_id="21m00Tcm4TlvDq8ikWAM"):
    url = f"https://api.elevenlabs.io/v1/text-to-speech/{voice_id}"
    headers = {"xi-api-key": api_key}
    data = {
        "text": text,
        "model_id": "eleven_multilingual_v2",
        "voice_settings": {
            "stability": 0.5,
            "similarity_boost": 0.5
        }
    }
    response = requests.post(url, json=data, headers=headers)
    return response.content
```

## 🔧 进阶技巧

### 1. 批量生成

```bash
# 生成多个版本对比
for voice in zh-CN-XiaoxiaoNeural zh-CN-YunjianNeural; do
    python3 tts_generator.py --script "文案" --voice $voice --output ${voice}.mp3
```

### 2. 调整语速（Edge-TTS）

```python
from edge_tts import Communicate

# 速度范围: -50% 到 +100%
communicate = Communicate("文案", "zh-CN-XiaoxiaoNeural", rate="+20%")
```

### 3. 添加停顿

```bash
# 用<break>标签添加停顿
python3 tts_generator.py --script "这是第一句话。<break time=\"500ms\"/>这是第二句话。"
```

## ❓ 常见问题

### Q1: Edge-TTS免费吗？
**A**: 完全免费。使用微软Edge浏览器的TTS服务，无限制。

### Q2: 可以商用吗？
**A**: 
- Edge-TTS: 需遵守微软服务条款，一般可用
- ChatTTS: 开源，可商用
- ElevenLabs: 付费版可商用

### Q3: 没有NVIDIA显卡能用ChatTTS吗？
**A**: 可以但很慢。建议用CPU模式或选择Edge-TTS。

### Q4: 声音可以克隆吗？
**A**: 
- Edge-TTS: 不支持
- ChatTTS: 支持，需训练
- GPT-SoVITS: 支持，5秒克隆
